> 如果好用就star和提bug呗~~

## Use

```javascript
/**
 * selector 选择器，只要是querySelectorAll支持的都可以
 * deep 是否强制完美对齐，如果不对齐则会有一定的透气空间
 **/
hyphen(selector, deep);
```
